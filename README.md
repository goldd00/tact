![Static Badge](https://img.shields.io/badge/npm-v9.6.4-red?logo=npm)

TactのWordpressテーマ

## 使用方法
```bash
git clone https://github.com/goldd00/tact
cd tact/
npm install
```

## 注意点
webpackを使用しているのでsass、もしくはjsを変更後は下記を必ず実行
```bash
npm run build
```
